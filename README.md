# MERN Task Tracker

A full-stack Task Tracking web application built as part of my internship assignment.

## 🚀 Tech Stack
- **Frontend:** React.js (Vite)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB

## 📦 Project Structure
- `/backend` - Express API, Mongoose models, and database configuration.
- `/frontend` - React application with dynamic state management.

## 🛠️ How to Run Locally

### 1. Clone the repository
```bash
git clone https://github.com/Zypher90/task-tracker
cd task-tracker
```

### 2. Setup backend
```bash
cd backend
npm install
# Create a .env file and add your PORT and MONGO_URI
npm run dev
```

### 3. Setup frontend
```bash
cd ../frontend
npm install
npm run dev
```
